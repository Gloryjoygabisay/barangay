# This is how to code in this app

You can open the terminal with the command `^ CMD t`
And in the terminal you can run the following commands:
- `npm run build` to see if it is building OK
- `npm run dev` to run it locally and see it show a link you can `CMD click` on to open in your web browser and see what the app looks like. You can enter `q` then enter in the terminal underneath where it says the browser link to stop the dev server running the game so you can continue working on the app.

You can open a very powerful copilot in the terminal by entering the command `copilot --yolo`
You can exit that copilot by entering `^ c` twice.
You can give just about any command to that yolo copilot and it will help you.

You can open copilot in vscode in the left menu area, or you can click the copilot chat icon at the top middle of vscode and select `new chat editor` and it will open in the main area.
